# Анализатор IP-строк (Dart)

Метод на Dart, которая анализирует строку с IP-адресами и подсчитывает количество каждого символа, с обработкой ошибок.

## Спецификация

- Язык : Dart (Нулевая Безопасность)
- Основной метод: 'analyzeIpString(ввод строки)`
- Возврат : 'Map<String, int>' с подсчетом допустимых символов (цифр и точек)


## Обработка ошибок

|       Случай        |           Выданное исключение              |
|---------------------|--------------------------------------------|
| Пустая строка       | `Exception("Строка не может быть пустой")` |
| Недопустимый символ | `Exception("Недопустимый символ 'x'")`     |


## Использование

```dart
  final analyzer = IpAnalyzer();
  final result = analyzer.analyzeIpString("192.168.1.1 10.0.0.1");
  // {"1":6, "9":1, "2":1, ".":6, "6":1, "8":1, "0":3}
```
